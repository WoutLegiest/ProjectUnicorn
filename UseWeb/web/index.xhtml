Vibration Project

Welkom op de home pagina van het vibration project.
Dit project bestaat uit twee verschillende onderdelen.

Het eerste onderdeel is de vibration spotter: Een applicatie gebouwd voor professionele gebruikers die niet aan de verleiding kunnen weeerstaan om overal trilling te registreren. De website biedt voor deze gebruikers, onder de menu optie "spotter", een hanidge tool met een overview van zowel zelf als door andere uitgevoerde metingen.

Het tweede onderdeel is het STEM-gedeelte: een applicatie gebouwd voor studenten die ermee kunnen onderzoeken welke invloed de structuur van een gebouw heeft op de trillingen. De website biedt voor de leerkracht een handige ondersteuning en geeft de mogelijkheid om onder andere projecten aan te maken en resultaten te bekijken.

Wij wensen u een prettig verblijf op onze site.

Dit project is met liefde gmeaakt door Pelle Reyniers, Boris Wauters en Wouter Legiest.
